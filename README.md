
## Previsão de Desempenho Acadêmico
O projeto consiste na aplicação de Aprendizado de Máquina Supervisionado para prever o desempenho acadêmico de alunos, usando o Student Performance Dataset (UCI).

Tipo: Supervisionado (Regressão / Classificação)

Base de Dados: Student Performance Dataset (UCI)

Descrição: Modelo que estime notas finais de alunos com base em fatores como estudo, absenteísmo e histórico escolar.
